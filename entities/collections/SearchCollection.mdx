Search Collections save each sub document in a vector database, enabling semantic search on the documents within the collection.


# Code Snippets
## Creating a Search Collection
```py
col = SearchCollection.create(
    name="Name Of Collection",
    model="instruct-xl",
    ttl=24
)
```

## Searching
Will return sub documents that are relevant to the query.

for more info read [Search Retrieval](</retrieval/Search>)

```py
await col.search("SEC Filings Mentioning Climate Change")
```

## Filtering a Collection
returns all context from page 1 for all documents in a collection

```py
await col.filter(page=1).search("who is the author of the paper?")
```
