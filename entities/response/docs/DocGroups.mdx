documents grouped based off of the group value passed to a retriever.

You can group by all, collection, document and soon by metadata.

For more information on grouping read [Retrieval Grouping](</retrieval/Retrieval#grouping>)





### Examples
- group every 2 pages in a pdf
- group every 300 seconds in a transcript
