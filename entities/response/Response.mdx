---
title: Response Model
description: EzLLM Response Model
---

the Response allows you to easily interact with the output of
1. Filter
2. Retrievals
3. Methods

The ResponseModel has properties and a standard format similar across all response types

```py
ResponseBase(
    costs=ResponseCosts(
        ResponseCost(
            cost=0.01061,
            limitId='gpt-4',
            ...
        )
    )
    docs=ResponseDocs(
        ResponseDoc(
            name='Doc Name'
        )
    )
    doc_groups=DocOutputGroups(
        OutputGroup(
            type='collection'
            id='col_id',
            data=ResponseDocs(
                ...
            )
        )
    )
)

MethodBase(
    ... # MethodBase extends ResponseBase
    output: MethodOutput(
        groups: OutputGroups(
            OutputGroup(
                type='collection',
                id='col_id',
                data=OutputData
            )
        )
    )
)
```
