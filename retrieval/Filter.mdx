title: Filter
description: EzLLM Filter Method 
---

The filter method enables you to filter on, documents, documents metadata, sub documents and collections.

Filter is inclusive, so if you filter on a collection and a document, it will include both in the response


# Code Snippets
```py
retrieval = ezllm.filter(
    documents=[doc, "658b4e441f41ead7592562de"],
    collections=[col, "658b4e441f41ead7592562de"],
)

print(retrieval)
```

### Props
<ParamField path="documents" type="(string | Document)[]">
    List of `Document` or document ids to include in filter
</ParamField>

<ParamField path="collections" type="(string | Document)[]">
    List of `Collection` or collection ids to include in filter
</ParamField>
